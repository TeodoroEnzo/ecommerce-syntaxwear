# SyntaxWear - Tênis e Sneakers Online

A **SyntaxWear** é uma landing page moderna e responsiva para um e-commerce especializado em tênis e sneakers. O projeto foi desenvolvido com foco em design limpo, navegabilidade e organização modular do código CSS.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada utilizando:
  - Variáveis CSS (Custom Properties).
  - Flexbox e CSS Grid para layouts responsivos.
  - Arquitetura modular de componentes.
  - Media Queries para adaptação em diferentes dispositivos.
- **Google Fonts**: Utilização da fonte 'Ubuntu' para uma tipografia moderna.

## 📦 Estrutura do Projeto

O projeto segue uma estrutura organizada para facilitar a manutenção:

```text
ecommerce-syntaxwear/
├── index.html          # Página principal do projeto
├── css/                # Diretório de estilos
│   ├── reset.css       # Reset de estilos padrão do navegador
│   ├── variable.css    # Definição de cores e fontes globais
│   ├── base.css        # Estilos base e utilitários (botões, etc.)
│   └── components/     # Estilos específicos por componente
│       ├── header.css
│       ├── hero.css
│       ├── product-category.css
│       ├── product-grid.css
│       └── footer.css
├── images/             # Ativos visuais (banners, ícones, produtos)
└── README.md           # Documentação do projeto
```

## ✨ Funcionalidades

- **Design Responsivo**: Adaptado para dispositivos móveis e desktop.
- **Navegação Intuitiva**: Menu de categorias (Masculino, Feminino, Outlet) e acesso rápido à conta e carrinho.
- **Seção Hero**: Banner principal impactante com chamadas para ação (CTA).
- **Categorias em Destaque**: Cards visuais para acesso rápido a estilos (Casual, Esporte, Moderno, Futurista).
- **Grid de Produtos**: Exibição organizada dos principais modelos de sneakers.
- **Newsletter**: Rodapé com formulário de inscrição e links para redes sociais.

## 🛠️ Como Visualizar

1. Clone este repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
